### CPagination

#### Props

| Prop name          | Description                                 | Type   | Values                         | Default |
| ------------------ | ------------------------------------------- | ------ | ------------------------------ | ------- |
| <code>align</code> | Set the alignment of pagination components. | string | `'start'`, `'center'`, `'end'` | -       |
| <code>size</code>  | Size the component small or large.          | string | `'sm'`, `'lg'`                 | -       |
