### CPagination

#### Props

| Prop name | Description                                 | Type   | Values                         | Default |
| --------- | ------------------------------------------- | ------ | ------------------------------ | ------- |
| **align** | Set the alignment of pagination components. | string | `'start'`, `'center'`, `'end'` | -       |
| **size**  | Size the component small or large.          | string | `'sm'`, `'lg'`                 | -       |
